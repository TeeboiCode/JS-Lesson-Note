JavaScript Scope

Scope refers to where variables, functions, and objects are visible and can be used in your code.

Global Scope

const appName = "TodoApp";  // global — visible everywhere

function showApp() {
  console.log(appName);  // ✅ works
}

Function Scope

function demo() {
  let secret = "hidden";
  console.log(secret);  // ✅
}
// console.log(secret);  // ❌ Error — not accessible outside

Block Scope

if (true) {
  let temp = 10;
  console.log(temp);  // ✅
}
// console.log(temp);  // ❌ Error

for (let i = 0; i < 3; i++) {
  console.log(i);  // ✅ inside loop
}
// console.log(i);  // ❌ Error — let is block-scoped
⚠️
var vs let

var ignores block scope and leaks out of if and for blocks. Always prefer let and const.
