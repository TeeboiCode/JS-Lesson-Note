Lesson 08

JavaScript Arrays

Ordered lists of any data type

What Is an Array?

An array is an ordered list of values. Use arrays when order matters — shopping cart items, quiz scores, or a list of names.

Creating Arrays

const numbers = [10, 20, 30];              // array of numbers
const names = ["Ada", "Emeka", "Chi"];     // array of strings
const mixed = [1, "hello", true, null];    // mixed data types
const matrix = [[1, 2], [3, 4]];           // array of arrays
const users = [{ name: "Ada" }, { name: "Bob" }];  // array of objects

Accessing Elements

console.log(numbers[0]);   // 10 — first element
console.log(numbers[2]);   // 30 — third element
console.log(numbers.length); // 3 — number of items

Modifying Elements

numbers[1] = 99;
console.log(numbers);  // [10, 99, 30]

Array Methods: push, pop, shift, unshift

const fruits = ["Apple", "Banana"];

fruits.push("Mango");      // add to END → ["Apple", "Banana", "Mango"]
fruits.pop();              // remove from END → ["Apple", "Banana"]
fruits.unshift("Grape");   // add to START → ["Grape", "Apple", "Banana"]
fruits.shift();            // remove from START → ["Apple", "Banana"]

Looping Through Arrays

const colors = ["red", "green", "blue"];

// Classic for loop with index
for (let i = 0; i < colors.length; i++) {
  console.log(i, colors[i]);
}

// for...of — cleaner
for (let color of colors) {
  console.log(color);
}

// forEach method
colors.forEach((color, index) => {
  console.log(index, color);
});

More Useful Methods

const scores = [85, 90, 78, 92];

scores.includes(90);                    // true
scores.indexOf(78);                     // 2
const doubled = scores.map(n => n * 2); // [170, 180, 156, 184]
const passed = scores.filter(n => n >= 80); // [85, 90, 92]
const total = scores.reduce((sum, n) => sum + n, 0); // 345
